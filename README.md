## Описание
[Чат (Slack)](https://slack-chat.up.railway.app) – Аналог Slack-чата, упрощенная версия.

## Требования

- Git клиент
- Node.js 18 или выше
- Make

## Установка

* Клонировать проект
```
$ git clone 
```

* Установить зависимостей
```
$ make install
```

## Использование

### Запуск для локальной разработки

```
$ make develop
```

### Сборка

```
$ make build
```

### Запуск собранного приложения

```
$ make setup
```
